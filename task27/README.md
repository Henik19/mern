#  Shoe Store React App

##  Project Overview

This is a simple React application for an online shoe store.
Users can view different shoes, add them to the cart, increase/decrease quantity, and see the total price.

---

##  Features

* 🛍️ View list of shoes
* ➕ Add items to cart
* 🔼 Increase quantity
* 🔽 Decrease quantity
* 🧮 Automatic total calculation
* 🛒 Shopping cart system
* 🎨 Attractive brown-themed UI
* 📱 Simple and responsive layout

---

##  Technologies Used

* React JS
* JavaScript (ES6)
* HTML
* CSS (Inline Styling)
* useState Hook

---

## Project Structure

src/

* App.js → Main component (UI + logic)
* index.js → Entry point

---

## How to Run the Project

1. Install dependencies:
   npm install

2. Start the project:
   npm start

3. Open browser:
   http://localhost:3000

---

## How It Works

* The app uses `useState` to manage the cart.
* When user clicks "Add to Cart", the item is added.
* If item already exists, quantity increases.
* User can increase (+) or decrease (−) quantity.
* Total price is calculated using `reduce()` function.
