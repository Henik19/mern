Laundry Wallah / Laundry Mart - Landing Page
A simple, semantic HTML-only landing page developed as part of a web development task. This project showcases basic HTML structure including tables, forms, and semantic elements without the use of CSS or <div> containers.

Project Overview
The objective of this project was to create a functional business landing page for a laundry service using pure HTML. The layout follows a specific structure requested in the task brief to demonstrate proficiency in core web elements.

Features
The page includes the following sections:

Header
About Section
Image Section
Services Section
Price list
Footer
