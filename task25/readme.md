# FlowApp Landing Page

This project is a modern SaaS landing page built using HTML and Tailwind CSS. The goal was to create a clean, visually engaging interface with smooth animations and a professional layout, similar to real-world product websites.

## Features

* Fixed glass-style navigation bar
* Hero section with animated text and background effects
* Feature section using a bento-style grid layout
* Testimonials with glassmorphism cards
* Gradient call-to-action section
* Multi-column responsive footer

## Design Approach

The main focus was to keep the UI modern but simple. Instead of using heavy libraries or external CSS, everything is handled using Tailwind utility classes. Small animation effects like hover scaling, floating elements, and fade-ins are used to make the page feel interactive.

## Technologies Used

* HTML5
* Tailwind CSS
* Custom Tailwind configuration for animations

## Responsiveness

The layout is responsive and adjusts across different screen sizes. Flexbox and grid are used to handle spacing and alignment, and Tailwind breakpoints are applied for mobile and tablet views.

## Animations

Basic animations are added using custom keyframes inside the Tailwind config. These include:

* Slide up effects
* Floating elements
* Hover scale transitions
* Glow effect on buttons

## How to Run

1. Download or clone the project folder
2. Open the `index.html` file in any browser
3. No installation or setup is required
