
## Project Objective

The main goal of this project is to understand how JavaScript functions work with arrays.  

In this project, we created different types of functions that:

- Find the maximum number in an array
- Calculate the sum of all elements
- Count how many odd numbers are present

This project is built using simple HTML, CSS, and JavaScript.


## Technologies Used

- HTML (Structure)
- CSS (Basic Styling)
- JavaScript (Logic & Functions)

## What I Practiced in This Project

In this project, I practiced:

- Working with arrays
- Using loops (`for` loop)
- Writing different types of functions:
  - Normal Function
  - Anonymous Function
  - Arrow Function
- Updating content using `innerHTML`
- Displaying results in both console and webpage
