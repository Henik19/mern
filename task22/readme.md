# Hero Landing Page

This is a simple hero section page I made using HTML and Tailwind CSS.

The idea was to create a clean landing page like modern tools (Notion type UI) where there is a heading, buttons, and some floating cards.


## What I added

* Sticky navbar on top
* Heading and small description
* Two buttons (Get AI free, Request demo)
* Image in center
* Some cards placed on image using position
* Chat button at bottom right


## Tools used

* HTML
* Tailwind CSS 

## What I learned

* How flex works for layout
* How to center things
* Difference between relative and absolute (this was confusing at first)
* How to use Tailwind classes instead of writing CSS

## Problems I faced

* Positioning cards correctly was difficult
* Image size was too big first, then I fixed height
* Understanding spacing in Tailwind took time

## How to run

Just open the HTML file in browser.
Internet is needed for Tailwind.
