# Animations & UI Feedback Demo

## Project Overview

This project is a simple demonstration of how animations and UI feedback can improve the user experience in modern web applications. The page is built using Tailwind CSS and focuses on interactive elements like buttons, cards, loading states, and notifications.

The main goal of this project is to understand how to use built-in Tailwind animation utilities along with small custom animations to make the interface more engaging and responsive.


## Features Included

### 1. Built-in Animations

This section shows basic Tailwind animations:

* Spinner (loading effect)
* Ping (notification pulse)
* Pulse (skeleton loader)
* Bounce (attention indicator)

### 2. Button Animations

Different button interactions are implemented such as:

* Scale effect on hover
* Shadow and scale combination
* Background color transition
* Rotation effect
* Lift (translate) effect
* Combined hover effects
* Click (active) feedback

### 3. Card Animations

Cards respond to hover actions with:

* Lift and shadow effect
* Scale and slight rotation
* Border reveal animation

### 4. Notification Badges

Interactive notification icons with:

* Animated dots
* Ping, pulse, and bounce effects
* Proper positioning using relative and absolute

### 5. Loading States

Different loading UI patterns:

* Skeleton loader
* Button with spinner
* Bouncing dots loader

### 6. Staggered Animations

Cards appear with a fade-in animation using:

* Custom keyframes
* Different delay timings (0ms, 150ms, 300ms, 450ms)


## Technologies Used

* HTML5
* Tailwind CSS


## How to Run the Project

1. Download or extract the project folder
2. Open the `index.html` file in your browser
3. Interact with elements (hover, click) to see animations

* How to design interactive UI components
