# AppName Landing Page

This project is a responsive landing page built using HTML and Tailwind CSS. It is designed for a SaaS platform called “AppName” that helps teams collaborate, build, and launch products efficiently.

The main goal of this project is to understand how responsive design works using Tailwind’s utility classes without writing custom CSS.



## Features

- Responsive navigation bar with mobile menu
- Hero section with flexible layout (column on mobile, row on desktop)
- Call-to-action buttons
- Features section using grid layout
- Clean and simple footer
- Fully responsive on mobile, tablet, and desktop

## Technologies Used

- HTML
- Tailwind CSS  


## Responsive Design Approach

This project follows a mobile-first approach using Tailwind CSS breakpoints:

- Default (Mobile)
- sm: Small devices
- md: Tablets
- lg: Laptops
- xl: Large screens

Flexbox and Grid are used to create layouts that adapt smoothly across different screen sizes.


## How to Run the Project

1. Download or clone the project folder  
2. Open the folder  
3. Double click on `index.html`  
4. The website will open in your browser
