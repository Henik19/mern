# Omnifood Website

## About Project

This is a simple food delivery website called **Omnifood**.
I made this project using only **HTML and CSS**.

It is a **single page website** and it works on mobile, tablet and desktop.

## What is inside website

### Navbar

* Logo on left side
* Menu links on right side
* Hamburger menu for small screens


### Hero Section

* Big heading text
* Two buttons:

  * I'm Hungry
  * Show me more


### How It Works

There are 3 steps:

1. Choose your plan
2. Pay and download app
3. Start ordering


### Cities Section

* 4 cities shown
* Each city has image and details


### Testimonials

* 3 customer reviews
* With images and text


### Pricing Section

* 3 plans:

  * Premium
  * Pro
  * Starter


### Footer

* Links
* Social icons


## Responsive Design

Website works on all screen sizes:

* Desktop → everything in row
* Tablet → 2 columns layout
* Mobile → single column layout

## Files in Project

* index.html → main file
* style.css → all styling
* images folder → all images


## How to run

1. Open project folder
2. Open index.html
3. Website will open in browser
